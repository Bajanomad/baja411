# Baja411

Baja411 is a map-first Baja California Sur field utility.

## App root

The Next.js app now lives at the repository root.

- Routes: `app/`
- Components: `components/`
- Prisma: `prisma/`
- Docs: `docs/`

## Validation

Run from repo root:

```bash
npm run dev
npm run lint
npm run build
```

Do not use `` commands.

GitHub is source of truth. Vercel is deployment path and should build from repo root unless dashboard settings override it.
