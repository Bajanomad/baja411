# Baja Sur 411 Project Guidelines

## Operating hierarchy

1. User first.
2. Engineer second.
3. CEO third.

## Repository role

This repository is the future platform repo for Baja Sur 411.

## Migration target

The current app migration target is `apps/web`.

## Native app status

`apps/ios` and `apps/android` are placeholder folders only at this stage.

## Deployment guardrail

Production deployment must not be switched until migration validation passes.
