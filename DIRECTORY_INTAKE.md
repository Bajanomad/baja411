# Baja411 Local Directory Intake Guide

## 1. Purpose

Baja411 helps locals, travelers, expats, road users, and people living in Baja find useful services, offices, resources, and local help quickly.

## 2. Core principle

Collect practical field use information first.
Pretty marketing copy comes later.

- Town
- Area / neighborhood

- Location notes

- WhatsApp
- Website

- Short description

- Cash only / card accepted if known
- Delivery available if relevant
- Service area if relevant
- Verified
- Verified by
- Last verified date
- Source notes
- Map pin status

## 6. Collection rules

- Prefer verified local info over random guesses.
- Mark uncertain data clearly.
- Do not invent phone numbers, hours, services, or coordinates.
- If a listing is useful but not verified, keep it as unverified.
- Call and WhatsApp links are more important than pretty marketing copy.
- Water, propane, mechanics, medical, fuel, roadside help, and emergency resources are high priority.
- Do not collect private personal info unless it is already intended as public business or service contact info.
- If a source is a Facebook comment, capture the suggested category and details but do not publish the commenter’s private profile info.

## 7. Verification levels

- unverified
- community suggested
- contact checked
- location checked
- fully verified

## 8. Publishing rule

This intake guide does not publish anything to the live app by itself.
Listings should only move into app data after review.
